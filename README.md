# Intent로 화면 전환
count button click -> 숫자 증가  
random button click -> 화면을 전환하며 0 ~ 현재 숫자 사이 랜덤한 값 출력  
  
원래 화면으로 되돌아 올 때 랜덤 값에서부터 시작
